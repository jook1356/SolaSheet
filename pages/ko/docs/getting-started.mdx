# 시작하기 (Getting Started)

PureSheet를 프로젝트에 설치하고 기본적인 그리드를 띄우는 방법을 알아봅니다.

## 설치 (Installation)

npm 또는 yarn을 사용하여 패키지를 설치합니다.

```bash
npm install puresheet
# 또는
yarn add puresheet
```

## 기본 사용법 (Basic Usage)

HTML 파일에 그리드가 렌더링될 컨테이너를 준비하고, 자바스크립트에서 `PureSheet` 인스턴스를 생성합니다.

```html
<div id="grid-container" style="height: 500px; width: 100%;"></div>
```

```javascript
import { PureSheet } from 'puresheet';
import 'puresheet/dist/puresheet.css'; // 스타일 시트 임포트

const data = [
  { id: 1, name: 'Alice', age: 25, role: 'Developer' },
  { id: 2, name: 'Bob', age: 30, role: 'Designer' },
  { id: 3, name: 'Charlie', age: 28, role: 'Manager' },
];

const columns = [
  { field: 'id', header: 'ID', width: 50 },
  { field: 'name', header: '이름', width: 100 },
  { field: 'age', header: '나이', width: 80 },
  { field: 'role', header: '직군', width: 150 },
];

const grid = new PureSheet('#grid-container', {
  data: data,
  columns: columns,
});

// 그리드 렌더링
grid.render();
```
