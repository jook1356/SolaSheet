# API Reference

PureSheet의 주요 API 명세입니다.

## Grid Options

`new PureSheet(container, options)` 생성자에 전달되는 옵션 객체입니다.

| 옵션명 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `data` | `Array` | `[]` | 그리드에 표시할 데이터 배열입니다. |
| `columns` | `Array` | `[]` | 컬럼 정의 배열입니다. |
| `rowHeight` | `number` | `35` | 행의 높이(px)입니다. |
| `headerHeight` | `number` | `40` | 헤더의 높이(px)입니다. |

## Column Definitions

`columns` 배열의 각 항목에 대한 설정입니다.

| 속성명 | 타입 | 설명 |
| :--- | :--- | :--- |
| `field` | `string` | 데이터 객체의 키(key)와 매핑됩니다. |
| `header` | `string` | 헤더에 표시될 텍스트입니다. |
| `width` | `number` | 컬럼의 너비(px)입니다. |
| `formatter` | `function` | 셀 값을 커스터마이징하는 함수입니다. |

## Methods

### `grid.render()`
그리드를 DOM에 렌더링합니다.

### `grid.setData(data)`
그리드의 데이터를 업데이트합니다.
