# PureSheet Documentation Site

This is the documentation website for [PureSheet](https://github.com/jook1356/PureSheet), built with [Next.js](https://nextjs.org) and [Nextra](https://nextra.site).

## Getting Started

### Prerequisites

- Node.js 18+ installed

### Installation

```bash
npm install
```

### Local Development

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

To build the static HTML export (for GitHub Pages):

```bash
npm run build
```

The output will be in the `out` directory.

## Project Structure

- `pages/`: Documentation pages (Markdown/MDX)
- `theme.config.jsx`: Theme configuration
- `next.config.js`: Next.js configuration
- `public/`: Static assets
